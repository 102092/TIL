# BFF란?

- `Backed For Frontend`
- 프론트엔드(웹, 모바일) 을 위한 맞춤형 API 를 제공하는 방식을 의미
- 하나의 백엔드 시스템에서 여러 프론트엔드를 지원하는 것이 아닌, 각 프론트엔드마다 필요한 데이터와 기능만을 제공하는 백엔드를 구성한다는 뜻
