# FQDN?
- Fully Qualified Domain Name
- 전체 도메인 이름을 표현하는 방식
- 호스트명부터, 최상위 도메인까지 모든 계층을 포함하고
- 각 레벨은 `.` 으로 구분됨.

## 예시
```txt
instance-3.asia-northeast1.compute.google.com.
```
- [`서비스/호스트명`].[`부서/지역`].[`서비스종류`].[`조직도메인`].[`최상위도메인`].
- 마지막 `.` 은 생ㄹ략 가능
